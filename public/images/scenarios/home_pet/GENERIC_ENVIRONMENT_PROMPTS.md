# Generic Environment Images - Comic Book Style

## Style Guide
All images should be:
- **Comic book illustration style** - bold lines, vibrant colors, clear panels
- **Environment-focused** - show the setting, not characters
- **No people or animals** - just the environment and objects
- **Consistent color palette** - warm, friendly, child-appropriate
- **Clean and simple** - focus on key elements

---

## Tips for Generation

1. **Consistent Style:** Use the same "comic book illustration style" description for all
2. **Color Mood:**
   - Morning: Cool, fresh blues and whites
   - Afternoon: Warm golden tones
   - Evening: Warm sunset colors
   - Success: Bright, vibrant
   - Failure: Muted, sadder tones
3. **Composition:** Keep it simple, focus on key objects (bowls, table, door, etc.)
4. **Atmosphere:** Let lighting and objects tell the story
