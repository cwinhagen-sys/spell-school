# Spell School Background Image Specifications

## Welcome Section Background Images

### File Names
Create 4 background images with these exact names:
- `spell-school-bg-1.png`
- `spell-school-bg-2.png`
- `spell-school-bg-3.png`
- `spell-school-bg-4.png`

### Dimensions
**Recommended size: 1400px × 400px**

**Why these dimensions?**
- The welcome section has a max-width of 1280px (max-w-7xl)
- With padding (px-6 = 24px each side), content width is ~1232px
- Height varies but is typically ~200-250px on desktop
- Using 1400px width ensures coverage on larger screens
- 400px height provides good coverage with aspect ratio 3.5:1

### Design Guidelines
- **Style**: Cool, magical spell school-themed illustrations
- **Opacity**: Images will be displayed at 20% opacity over the gradient
- **Colors**: Should complement the purple-indigo-pink gradient theme
- **Content**: Magical elements like:
  - Floating spell books
  - Glowing runes or symbols
  - Magical sparkles/stars
  - Wizard staffs or wands
  - Mystical portals
  - Arcane patterns

### Technical Details
- **Format**: PNG (with transparency) or JPG
- **File Location**: `/public/assets/spell-school-bg-1.png` through `spell-school-bg-4.png`
- **Rotation**: Images change every 60 seconds automatically
- **Transition**: Smooth 1-second fade between images

### Canvas Setup in Canva
1. Create custom size: **1400px × 400px**
2. Use spell school color palette (purples, indigos, pinks)
3. Design should be subtle enough to work at 20% opacity
4. Export as PNG (for transparency) or JPG
5. Save as `spell-school-bg-1.png`, `spell-school-bg-2.png`, etc.

### Example Themes for Each Image
- **Image 1**: Floating spell books and magical scrolls
- **Image 2**: Glowing runes and arcane symbols
- **Image 3**: Wizard staffs and magical sparkles
- **Image 4**: Mystical portal or magical energy waves
